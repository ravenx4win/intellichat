# Intellichat - Document Q&A Assistant
# Simple Commands to Run the Project

## 🚀 QUICK START (One Command):
```bash
cd /Users/mac/Documents/Intellichat && source venv/bin/activate && streamlit run app.py --server.port 8502
```

## 📋 What This Does:
- Navigates to project directory
- Activates virtual environment  
- Starts the application on port 8502
- Opens browser to http://localhost:8502

## 🆓 FREE Features:
- Uses Hugging Face API (completely free)
- No billing or quota issues
- Upload PDF documents
- Ask questions in natural language
- Get intelligent answers
- Maintain conversation history

## 🔧 Setup (First Time Only):
1. Get free API key: https://huggingface.co/settings/tokens
2. Add to .env file: HUGGINGFACE_API_KEY=your_key_here
3. Install dependencies: pip install -r requirements.txt
4. Initialize database: python init_db.py

## 🌐 Access:
- Open: http://localhost:8502
- Enter Hugging Face API key in sidebar
- Upload PDF and start chatting!

## 🆘 Troubleshooting:
- Port busy? Use --server.port 8503
- Missing deps? Run: pip install -r requirements.txt
- API issues? Check .env file
- Database issues? Run: python init_db.py check
